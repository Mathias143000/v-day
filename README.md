# Интерактивная валентинка

Милая страница ко Дню святого Валентина: кнопка «Да» растёт после каждого нажатия на «Нет», GIF меняются по настроению, играет музыка, а в финале запускается конфетти.

## Что внутри

- Полная русская локализация интерфейса.
- Адаптивная верстка для телефона и десктопа.
- Нежный фон с плавающими сердечками.
- Переключатель музыки с доступной подписью для скринридеров.
- Финальная страница `yes.html` с праздничной анимацией.

## Запуск локально

Проект не требует сборки. Откройте `index.html` в браузере или запустите простой статический сервер из корня проекта:

```bash
python -m http.server 8080
```

После этого страница будет доступна по адресу:

```text
http://localhost:8080
```

## Структура

```text
v-day/
├── index.html       # Главная страница с вопросом
├── yes.html         # Финальная страница после ответа «Да»
├── script.js        # Логика главной страницы
├── yes-script.js    # Конфетти и музыка на финальной странице
├── style.css        # Стили и адаптивность
└── music/           # Фоновая музыка
```

## Настройка

- Текст вопроса меняется в `index.html`.
- Финальное сообщение меняется в `yes.html`.
- Фразы кнопки «Нет» и подсказки меняются в `script.js`.
- Цвета, размеры и анимации меняются в `style.css`.
- Музыку можно заменить файлом в папке `music/`, обновив путь в HTML при необходимости.

## Публикация на GitHub Pages

1. Откройте настройки репозитория на GitHub.
2. Перейдите в **Pages**.
3. Выберите источник **Deploy from a branch**.
4. Выберите ветку `main` и папку `/ (root)`.
5. Сохраните настройки.

Через пару минут сайт будет доступен по адресу GitHub Pages вашего репозитория.
